# 涌城 / TIDELINE

WMDC 2026 元创未来参赛项目：极端天气下的滨水街区数字孪生推演系统。

## 技术路线

- React + TypeScript
- Three.js / React Three Fiber
- 浏览器端实时三维渲染
- Sites 在线部署

## 开发约定

- 每完成一个独立功能，创建一次独立提交并推送到 GitHub。
- 三维场景、模拟逻辑、界面系统和参赛材料分别归档。
- 页面中的演示数据明确标注，不冒充真实监测数据。

## 功能里程碑

1. 三维滨水街区与镜头漫游
2. 晴天、小雨、暴雨动态天气
3. 内涝扩散与时间轴推演
4. 人群疏散和应急车辆调度
5. 风险图层、建筑信息与前后对比
6. 推演报告生成
